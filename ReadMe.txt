-------------------------------------------flash_real.exe----------------------------------------------------------------
1.选择相应的设备id，测试结果以设备id+时间 来命名保存
2.选择测试类型（只读、只写、读写，全部），默认测试所有
3.将测试的真实文件命名为test文件大小.rar，放在D盘flash_test_file文件夹中。 例如1G文件命名为test1024M.rar
3.根据放入的真实文件设置文件大小，点击确定后提交数据。 只识别数字，填入其他数据则不作处理。表格无需填满，也无需按顺序填
4.设置buffer大小，点击确定后提交数据。 只识别数字，填入其他数据则不作处理。表格无需填满，也无需按顺序填
5.点击开始测试即可，中途可以点击保存当前数据按钮来保存临时结果。测试完成后会自动保存结果。
